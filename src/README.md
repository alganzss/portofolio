# VYBE Streetwear E-Commerce

Website e-commerce streetwear modern dengan desain editorial yang bold dan rebellious.

## 🚀 Deploy ke Vercel

⚠️ **PENTING**: Project ini dibuat di Figma Make dan tidak memerlukan build process tradisional.

### Cara Deploy yang Benar:

**Opsi 1: Deploy Langsung dari Figma Make**
1. Klik tombol "Deploy" atau "Share" di Figma Make
2. Pilih Vercel sebagai platform deployment
3. Figma Make akan otomatis handle deployment

**Opsi 2: Deploy Manual via Vercel**
1. Export/Download project dari Figma Make sebagai static files
2. Upload ke Vercel melalui dashboard
3. Atau connect GitHub repository jika sudah di-export

### ⚠️ Catatan Penting

Project ini tidak memiliki `package.json` atau build configuration standar karena dibuat di Figma Make. Jika Anda mendapatkan error "No Output Directory", itu berarti:

- Figma Make environment menghandle bundling secara otomatis
- Tidak perlu menjalankan `npm run build`
- Deploy langsung dari Figma Make atau export terlebih dahulu
