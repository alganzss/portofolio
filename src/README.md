# VYBE Streetwear E-Commerce

Website e-commerce streetwear modern dengan desain editorial yang bold dan rebellious.

## 🚀 Deploy ke Vercel

### Cara 1: Deploy via GitHub

1. Push project ini ke GitHub repository Anda
2. Kunjungi [Vercel](https://vercel.com)
3. Klik "Add New Project"
4. Import repository GitHub Anda
5. Vercel akan otomatis mendeteksi setting (Vite framework)
6. Klik "Deploy"

### Cara 2: Deploy via Vercel CLI

```bash
# Install Vercel CLI
npm i -g vercel

# Login ke Vercel
vercel login

# Deploy
vercel
```

### Cara 3: Deploy Manual

1. Build project:
```bash
npm run build
```

2. Upload folder `dist` ke Vercel melalui dashboard

## 🛠️ Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## 🎨 Tech Stack

- React 18
- TypeScript
- Tailwind CSS v4
- Motion (Framer Motion)
- Vite

- Hero section dengan split layout
